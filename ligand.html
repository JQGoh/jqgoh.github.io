Ligands on a Cluster

Attaching Ligands to a Cluster Effectively

June 15, 2016

Ligands, molecules that are attached to a cluster core (group of atoms), are important to decide the cluster properties. The following has one example of a metallic cluster core - an icosahedral gold clusters.

Cluster science community has actively studied thiolates-protected gold clusters because of the interesting properties associated to gold clusters. A thiolate-ligand such as S(CH3)AuS(CH3) can be attached to the icosahedral core as shown below:

The ligand is attached to the gold surface by connecting their two sulfur atoms with two surface gold atoms. Attaching ligands to a cluster core manually by using a molecular visualizer (such as Jmol) with a computer mouse can be a repetitive and boring job.

Using the  Atomic Simulation Environment package, I designed a programme that can automate the process of attaching the ligands to the surface of a cluster. The automation process assumes that a ligand is attached to only two surface atoms, marked by the sites (green dots) as shown below. If a ligand has occupied the sites marked by green dots, another ligand will not be able to occupy the sites marked by red dots as it will overlap with the existing ligand.

The designed programme can

  1. Determine the surface atoms of a cluster core.
  2. Thereafter, it automatically attach a given ligand molecule (with pre-defined orientation) to the surface of the cluster.

Here is the result of attaching ligands to an icosahedral gold cluster using the programme. This helps to save my effort in studying the effect of ligands orientation on cluster cores of different sizes and shapes in the future. The example code of this post and the main programme are available on my GitHub page.
